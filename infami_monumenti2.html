Help

The application allows to browse different items somehow connected to the various meaning associated to the phallus across different cultures worldwide.

Choose a narrative from the table:

You can select items associated to the same time period, place or artistic genre from links in the table.

See Schema Ontology properties:

The same information displayed in the table has been mapped according to properties from Schema Ontology. You can see the matching Schema property by clicking on the features in the table. More information about Schema and the properties we selected can be found in the project documentation.

Select a narrative:

In order to make the browsing experience easier, times, places and genres to which the items in our collection are associated can be selected from the icons on the top.
