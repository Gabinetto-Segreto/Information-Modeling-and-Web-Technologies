INFAMI MONUMENTI Documentation

Project


Infami Monumenti is a project that seeks to present a History of Phallic Culture, thinking of it as the invocation of the phallus the source of an alleged masculine power. Therefore, it was designed as a virtual museum with objects that present this culture and its main concepts: power, fertility, fortune and strength. There are specific settings for the page that presents all the objects, entitled Museum and for the presentation of each object within the Narratives.

Layout


Graphically representing a story of Phallic Culture involved thinking about what the metaphors of this culture are. The verticality of objects that mimic the phallus in different cultures was the first starting point. It also represents, in some of the societies presented here, the notion of fertility, in a clear allusion to the growth of plants, animals and fortunes. To achieve this, the use of sidebars to guarantee access to content was prioritized. Another functionality explored was content scrolling. This required the use of wide margins for large screens, and dynamic layout programming that prevents text from being squeezed onto smaller screens.


Colors


As Priapus is the most recurrent Western representation over time of the concept of the phallus as a symbol of power, we chose the most famous image of this deity, the fresco from the Casa dei Vetti in Pompeii as the starting point for the overall color palette of this project. The colors were extracted using palette identification tools through images such as those offered by Canva and Adobe Color. Thinking of the website as a virtual museum, neutral and soft colors were designed for the background. In contrast, the body texts were given bold colors.

Typography


As the project sought to portray the history of phallic culture in different contexts, the choice was made for a neutral font, avoiding interference in the presentation of the objects. At the same time, this font would have to have flexibility in its styles, allowing for some variation, allowing parts of the text to stand out. The choice fell on the Inter font. In the titles, the font has been used in its ExtraBold 800 variation and in the text in Light 300.


Museum


The organization of items to tell the History of Phallic Culture was thought of as a museum exhibition. One of the inspirations was the Gabinetto Segreto of the National Archaeological Museum of Naples. This section of the MAN contains, since the time of the Bourbon sovereigns of Naples, all works considered erotic or obscene, including some of the testimonies of phallic culture that were selected in this project.


Layout


Trying to reproduce the feeling of starting a visit to a museum, we sought a different layout on this page. By placing the museum plan occupying the entire page, the desire was to prevent the user from having any suggestion of what their visit will be like, seeking to reproduce the feeling of the visitor who finds himself at the entrance of a museum. Even though the navbar is available for the user to access objects in another way, the arrangement of the page's main content occupying the entire screen is an invitation for the user to explore a space freely. As in many museums, works are presented im rooms according to their type of production: architecture, literary works, paintings, sculptures and frescoes. To achieve this effect, the Museum page was diveded in different columns and rows to replicate the structure of a museum plan. For this, a grid was used with columns of different sizes in each row. Then, photos of objects of each type were inserted into carousels, allowing the user to have a preview of the objects in each room.


Colors


To create the grid that mimics the floor plan of a museum, the color palette was slightly changed. As it was necessary to create a great contrast between each of the rectangles (the rooms), we chose a palette that is present in the excavations of Pompeii and found in different studies on color in statues of the period.

StoryMap


This interactive map, created with Knight Lab's StoryMap, traces the evolution of phallic culture through time and geography. Each slide highlights significant historical periods and locations, represented by the objects selected for this project. Custom markers and multimedia content enhance the narrative, offering viewers an engaging, educational experience. The choice for this tool between others was determined by the facts that is designed to be user-friendly, easily navigable, and shareable.


Layout


StoryMap features a clean, user-friendly layout designed for seamless storytelling. The main screen displays an interactive map alongside a narrative panel. Each "slide" in the story corresponds to a specific location on the map, highlighted as viewers navigate through the slides. The narrative panel contains multimedia content, including text, images, and videos, providing context and enhancing the story. Some media elements need to be programmed in HTML within the tool. Users can easily switch between slides using navigation arrows or a timeline at the bottom.


Colors


For the map styling we used Mapbox, a powerful mapping platform that enables developers to build and integrate custom maps and location-based services into applications. Offering robust APIs and SDKs, Mapbox provides detailed, interactive maps with customizable styles. Using the tool, we chose earthy tones from the tones already used on the other pages. This choice is related to the ancient belief that the phallus would be a symbol of fertility, including in agriculture, leading to the use of tones belonging to this universe.

Typography


While Mapbox offers a wide typographic variety, allowing the use of the same fonts that we use in the rest of the project, StoryMap does not. Despite offering a very extensive API, attempts to change the source proved unsuccessful, and according to Knightlab itself, this is a problem that has not yet been resolved in the tool's JavaScript. However, it was possible to find fonts with similar characteristics to those chosen on the rest of the page: for the titles the font is Raleway and for the other texts Bitter was used.


Timeline


The timeline covers a broad span of time, from Paleolithic Times to the XXI Century A.D., with entries for every object in the project.


Layout


The design inspiration revolves around fertility metaphors, linked to phallic culture, appearing as a tangled branch that occupies in waves the page (for 992 pixels or more screen-sizes) or verticaly (for smaller screens). To produce this effect, the navigation is visually enhanced with horizontal and vertical lines and rounded corners. The objective is emphasizing both historical significance and symbolic representation. The page culminates in a call to action with a button leading to the “Museum” section, which provides a more detailed virtual exploration of phallic cultural artifacts.

The page uses Bootstrap 5.3 for responsive design, ensuring compatibility across various devices and screen sizes.


Narratives


In order to represent the variety of objects associated to the theme of the exhibition, items were organized according to three main narratives: chronologically, spatially and by genre. The first narrative groups the objects according to the century they were created in, revealing the persistency of symbolic representations associated to fertility and virility throught time. The spatial narrative organizes items according to the location where the creation of the content took place and it is yet another way to observe the widespread cult of the phallus and the meanings associated to it worldwide. Finally, in the last narratives items are presented according to their genre, from visual artifacts - like paintings, frescos and sculptures - to literary works, to more recent experiments in architecture.


Features


Along with the option to switch narratives from the fields in the table, we decided to reintroduce the categories in which the items are organized in an offcanvas element, both to provide the user with an overview of the available options and to make navigation more immediate. The offcanvas can be toggled from a secondary navbar, that also includes a link to a modal window with some information on how to use the app.


For each item in our exhibition, variables in the table were mapped to properties from Schema Ontology. The best solution for displaying the Schema properties in the application seemed to be using popups, so that the basic properties are always visible, and the Schema properties — which may be of interest only to a specialized audience — can be activated when needed, thus ensuring the ability to view the mapped property without losing sight of the original property. With the premise that the number of properties varies depending on the item (for example, not all objects have an author, and it hasn't always been possible to associate an authority control), the selected properties are listed below, along with a brief description of its meaning and usage.

Layout


Starting from the article which inspired the title of the project, we tried to use strategies that could convey a sense of monumentality. Specifically, we organized the exhibition page in well defined horizontal and vertical areas, by meshing up Bootstrap components - such as the main navbar, the modal and the offcanvas elements - and CSS grid. The resulting layout is composed of three columns, one for each content available for the item: an image, some textual content and a table that gathers the main information about the object.

To avoid overcrowding the page and, at the same time, maintain the same design principle, for smaller screens we adopted a two-column layout, where the columns are preceded by the name of the item to which the contents refer. Textual content and image are displayed by default, while the table can be toggled by the user. Concurrently, the width of the offcanvas which controls the narratives selection was adjusted to convey a stronger sense of verticality by splitting the screen in half.
